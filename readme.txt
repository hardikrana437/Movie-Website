Fresh Tomatoes Movie Trailer

Description
A static web page that displays movie trailers and allows the user to click on each trailer and play it.

Installation
Python 2.7 needs to be installed to run the program

Working with classes 
Blueprint known as class of Movie is created
Instances of class are created to call methods and variable in class 

How to run the code
Download Python platform from link(https://www.python.org/downloads/)
Download Zip and extract to your local environment.
Inside the project folder (ie NanoDegree), from the idle terminal, run python entertainment_center.py
Screenshot of working of website is saved in folder as Capture

Python
